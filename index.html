Hexo 安裝步驟

Step1 利用NVM裝Node.js

打開terminal，貼上下列script，裝上NVM

$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.31.0/install.sh | bash

裝好NVM后要加入路徑才能在shell下使用nvm，輸入下列指令來加入。

$ echo "source $(brew --prefix nvm)/nvm.sh" >> .bash_profile

然後還是用source過才能運行

$ . ~/.bash_profile ，到此nvm安裝完成。

接著要利用nvm裝Node.js，先用nvm ls-rmote 看版本，然後選一個喜歡的版本安裝nvm install <version>，接下來就是無止境的等待，不知道為何npm 與 nvm 的安裝速度差距如此巨大。

PS. 如果要使用nvm 無痛換版，轉台這裡。

Step2 利用Homebrew 裝 Git

打開terminal，貼上下列script

$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/instal

使用Homebrew 裝git

$ brew install git

Step3 安裝Hexo

利用git --version檢查git是否正確安裝。

利用npm --version檢查node.js是否正確安裝。

確認都沒問題后執行npm install -g hexo-cli ，安裝后可以利用hexo -v 確認是否正確安裝了。都沒問題后基本上系統已經假設完成。

Step4 開個Blog專用Folder

開個Folder當Blog 的root（要全空的），接著執行

要更新Hexo可以使用npm update，要檢查package.json中配置的dependencies 版號可以使用npm ls --depth=0
